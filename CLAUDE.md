# Project instructions

This repo has the `ecc@ecc` plugin (affaan-m/ECC) registered in `.claude/settings.json`.

For all website development work in this repo (HTML/CSS/JS, layout, components, performance, accessibility, code review), prefer ECC's agents, skills, and commands over ad-hoc approaches when one fits the task. Fall back to default tools/behavior when no ECC component is a good match.
